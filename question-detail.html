How to join 2 columns in a data set to make a separate column in SQL

sql beginner
2 answers

I'm trying to create a new column in SQL that combines data from two existing columns. I have a table with first_name and last_name, and I want to create a full_name column that combines them. How can I do this in SQL? I'm new to SQL and would appreciate a beginner-friendly explanation.

Accepted Answer

In SQL, you can create a new column that combines data from two columns using the CONCAT function. Here's how you can do it:

                                    
ALTER TABLE your_table
ADD full_name VARCHAR(100);

UPDATE your_table
SET full_name = CONCAT(first_name, ' ', last_name);

This will create a new column called 'full_name' and populate it with the combined values from first_name and last_name.

Alternative Solution

You can also use the '+' operator to concatenate strings in SQL:

                                    
SELECT first_name + ' ' + last_name as full_name
FROM your_table;

This approach is simpler but doesn't create a permanent column in your table.
